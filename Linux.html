2.5.7.1 Linux

La boîte de dialogue Configuration de l'impression (Print Setup) vous permet de configurer l'utilisation de l'imprimante par l'application.

La fenêtre Mise en page contient :

Formaté pour (Format for) - une liste permettant de sélectionner une imprimante particulière. Les entrées de cette liste sont définies par les lignes printer du fichier ~/.xtrkcad/xtrkcad.rc. La première entrée, "File", permet d'envoyer les impressions dans un fichier. Vous serez invité à entrer un nom de fichier lors de l'impression.

Taille du papier (Paper size) - une liste des différents formats de papier pris en charge.

Orientation - un bouton pop-up pour sélectionner différents formats de Paysage ou Portrait.

Appliquer (Apply) - met à jour les données et ferme la boîte de dialogue.

Annuler (Cancel) - termine le dialogue sans mise à jour.
