2.9.1.1 Boite de dialogue nom de véhicule (Car Item Dialog)

Cette boîte de dialogue est appelée à partir des boîtes de dialogue Inventaire des véhicules (Car Inventory) (Section 2.9.1) et Gestion personnalisée (Custom Management) (Section 2.9.2) .

La boîte de dialogue peut être affichée de trois manières différentes et permet de créer ou de modifier les noms de véhicules, les pièces de véhicules ou les prototypes.

Le bouton Ajouter (Add) crée une nouvelle définition. La boîte de dialogue reste ouverte pour vous permettre d'entrer plusieurs définitions. Vous pouvez Fermer (Close) la boîte de dialogue lorsque vous avez terminé. Si vous modifiez une définition existante, le bouton Ajouter sera remplacé par le bouton Mise à jour (Update) qui met à jour la définition sélectionnée et ferme la boîte de dialogue.


Le bouton Nouveau (New) de la boîte de dialogue Nouveau véhicule appelle la boîte de dialogue Pièces ou Prototype que vous pouvez utiliser pour créer nouvelles définitions de pièces ou de prototypes pour vos voitures. Le bouton Nouveau de la boîte de dialogue Pièces appelle la boîte de dialogue Prototype.

Boîte de dialogue de noms de voiture

La boîte de dialogue contient :

Une description du véhicule contenant les listes Fabricant (Manufacturer), Prototype et Référence (Part Number), utilisées pour sélectionner une pièce détachée d'une voiture. La liste Fabricant (Manufacturer) contient des entrées Personnalisée (Custom) et Inconnue (Unknown) pour des pièces de voitures spéciales. Pour ces entrées, vous pouvez saisir les champs Référence N° (Part Number) et Description.

  • Si le fabricant de votre voiture ne figure pas dans la liste Manufacturer (Fabricant), vous pouvez entrer le nouveau nom. De même, si la Part (Pièce) ne figure pas dans la liste, vous pouvez entrer un nouveau numéro de pièce. La nouvelle pièce sera ajoutée à la liste des pièces détachées auto (Section 2.9.2).

Les boutons radio Mode vous permettent de choisir entre les contrôles Information décrits ici ou les contrôles Personnaliser (Customize) décrits dans la section suivante. Utilisez l'option Personnaliser (Customize) pour modifier le nom, le numéro ou la couleur de la voie si vous avez repeint la voiture ou en ajuster les dimensions.

Informations détaillées sur chaque élément de véhicule, telles que Index, Prix (Price) et État (Condition), Dates et Remarques (Notes). La valeur Index doit être unique pour tous les éléments de véhicule.

  • Pour entrer plusieurs Nouveaux véhicules (New Cars), vous pouvez définir le champ Quantité (Quantity) sur une valeur supérieure à un. Le groupe de boutons radio Numéros (Numbers) indique si les numéros du nouveau véhicule seront séquentiels (tels que 1234, 1235, ...) ou répétés.

Une ligne de message contiendra des instructions si des champs doivent être renseignés.
