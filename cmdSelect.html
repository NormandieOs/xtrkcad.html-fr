2.2.15 Sélectionner

Un clic sur le bouton Sélectionner annule toutes les commandes actives, ferme toutes les fenêtres de dialogue ouvertes et active la fenêtre principale.

En combinaison avec la souris, la commande Sélectionner peut être utilisée comme suit :

Clic gauche - sélectionne un objet (ou désélectionne un objet sélectionné). Les objets sélectionnés apparaissent en rouge (sauf modification dans la boîte de dialogue Couleur (Section 2.10.1)).

Maj + clic gauche - sélectionne toutes les voies non sélectionnées connectées à la voie sélectionnée. La sélection s'arrête sur les voies sélectionnées. Attention : Ceci peut sélectionner toutes les voies du plan.

Clic gauche + glisser - met en surbrillance une zone où tous les objets de cette zone sont sélectionnés.

Clic droit - affiche un menu contextuel de commandes.


ou

Maj + clic droit (Section 3.4) : affiche un menu contextuel d'options permettant de contrôler le dessin des objets lors des déplacements et des rotations. (au dessous de). S'il existe des objets sélectionnés, des éléments de menu supplémentaires vous permettent de faire pivoter les objets sélectionnés selon différents angles.

Clic droit + Glisser : met en surbrillance une zone et désélectionne tous les objets de cette zone.

Maj + clic gauche + glisser : déplace les objets sélectionnés (commande Déplacer (Section 2.2.8)).

Maj + clic droit + glisser : fait pivoter les objets sélectionnés (commande Rotation (Section 2.2.14)).
  • Si vous appuyez sur la touche Ctrl pendant le glissement, la rotation se produira par incréments de 15 °.

Ctrl + clic gauche + glisser : déplace les libellés d'aiguillage et d'élévation (voir également la commande de Description de déplacement (section 2.2.9) ).

Les commandes suivantes agissent sur les objets que vous avez sélectionnés :

  • Déplacer (Move) (Section 2.2.8) les objets sélectionnés.
  • Faire pivoter (Rotate) (Section 2.2.14) les objets sélectionnés.
  • Inverser (Flip) ( Section 2.2.5 ) (reflète) les objets sélectionnés.
  • Supprimer (Delete) ( Section 2.4.3 ) les objets sélectionnés.
  • Masquer (Hide) dans un tunnel (Section 2.4.4). ).
  • Déplacer (Move) au-dessus ou au-dessous (section 2.4.1).
  • Nomenclature (Parts list) (Section 2.9.6) des objets sélectionnés.
  • Modifiez (Change) la largeur des rails des voies.
  • Exporter (Section 2.5.1) les objets sélectionnés.
  • Créer une définition de structure ou d'aiguillage à partir d'objets Groupés (Grouped) (Section 2.9.4).
  • Modifier l'échelle (Rescale) ( Section 2.2.1 ) des objets sélectionnés.

Les commandes Couper (Cut), Copier (Copy) et Coller (Paste) trouvées dans Editer (Edit) (Section 2.4) fonctionnera également sur les objets sélectionnés.

Remarques :

  • Une forme pleine (Section 2.3) peut être sélectionnée en cliquant sur son bord.

  • Les connexions entre voies sélectionnées et non sélectionnées seront interrompues lorsque les voies sélectionnées seront déplacées. Si l'échelle du dessin est égale ou inférieure à 8:1, ces connexions sur le point d'être rompues sont signalées par un X rouge.

  • Vous pouvez utiliser la commande Tout sélectionner (Select All) du menu Modifier (Section 2.4) pour sélectionner tous les objets de la présentation.

  • Le bouton radio de commande Option par défaut (Section 2.10.) permet de contrôler si la Propriétés (Section 2.2.12) ou la commande Sélectionner est utilisée comme commande par défaut.

  • Appuyez sur la touche Echappement (Escape) pour désélectionner toutes les voies sélectionnées.
