2.9.1 Inventaire des véhicules (Car Inventory)

Cette boîte de dialogue permet de gérer votre liste de voitures et de locomotives.

La partie centrale de cette boîte de dialogue contient une liste détaillée de chaque voiture et locomotive de votre inventaire. Vous pouvez trier la liste sur différentes colonnes en sélectionnant leurs noms dans les Listes déroulantes en haut de la boîte de dialogue. La largeur des colonnes peut être ajustée avec Gauche+Glisser sur les lignes entre les titres de colonne.

Les voitures sont positionnées sur le plan à l'aide de la commande Trains (Run Trains) (Section 2.9.8).

Le bouton Rechercher (Find) centre la fenêtre principale (Section 3.1) sur la voiture sélectionnée. Ce bouton est activé uniquement pour les voitures qui se trouvent sur le plan (indiqué sur le Plan dans la colonne Emplacement). Si la voiture est sur une voie cachée avec le Trains sur une voie cachée (Section 2.10.3) qui est défini comme Masqué (Hide), alors la voiture (et toutes les voitures accrochées) sont rendues visibles.

Le bouton Editer (Edit) appelle la boîte de dialogue Mettre à jour le véhicule (Section 2.9.1.1) pour modifier la voiture sélectionnée. Ce bouton est désactivé si plus d'une voiture (ou aucune) est sélectionnée ou si la voiture sélectionnée est sur le plan.

Le bouton Ajouter (Add) permet d'ajouter une nouvelle voiture. Reportez-vous à la Section A.3 si une erreur se produit lors de la tentative d'ajout à cette liste d'inventaire.

Le bouton Effacer (Delete) supprime les voitures sélectionnées de votre inventaire. Ce bouton est désactivé si l'une des voitures sélectionnées se trouve sur le plan.

Les boutons Importation et Exporter lisent et écrivent dans un fichier Comma-Separated-Value (CSV). Ce format de fichier peut être lu et écrit par de nombreuses applications. Il peut être utilisé pour transférer des données d'inventaire de voiture entre différentes applications. Ce fichier contient les colonnes suivantes (les quatre premières sont obligatoires lors de l'importation) :

Scale - exemple : HO, HOn3, N, O etc

Manufacturer et Partno

Prototype - exemple : GP40

Index - un identifiant numérique unique (optionnel)

Roadname, Repmark et Number - exemple : Burlington Northern, BN, 1234

Type - entrez l'un des éléments suivants :
  • 10101(Diesel),
  • 10201(Steam),
  • 10301(Electric),
  • 30100(Freight),
  • 50100(Passenger),
  • 70100(M-O-W) ou;
  • 90100(Other)

Options - 2 (Locomotives), 4 (Body Mounted Couplers)

CarLength, CarWidth, CoupledLength and TruckCenter - dimensions des voitures en pouces

Color - codé comme (Red * 65535 + Green * 256 + Blue)

PurchPrice, CurrPrice, PurchDate et ServiceDate - valeurs numériques

Condition - entrez l'un des éléments suivants :
  • 100(Mint),
  • 80(Excellent),
  • 60(Good),
  • 40(Fair),
  • 20(Poor) or;
  • 0(N/A)

Notes - texte de forme libre

Le bouton Liste crée un fichier texte répertoriant votre inventaire.
