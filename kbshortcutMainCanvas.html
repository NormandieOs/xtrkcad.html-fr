3.3.8 Dessin principal (Main Drawing Canvas)

Dessin principal

Ces actions prennent effet si le plan du dessin principal dispose du "keyboard focus" activé (Le keyboard focus est une fonction système de l'ordinateur, une seule application à la fois peut en disposer).

Move Canvas:
  • Bas - Down Arrow
  • Gauche - Left Arrow
  • Droite - Right Arrow
  • Haut - Up Arrow

Fenêtre de navigation
  • On / Off - Ctrl+Space
