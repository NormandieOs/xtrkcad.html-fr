C.2.49 Etiquette de description cachée

La commande Déplacer la description ne peut être utilisée que lorsque des étiquettes sont visibles sur le plan. Les étiquettes peuvent être masquées à l'aide du menu contextuel.

Utilisez le menu contextuel pour activer et désactiver la visibilité d'une voie.
