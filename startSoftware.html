1.6 Démarrer le programme

L'exécutable XTrackCAD est nommé xtrkcad.exe sous Windows et xtrkcad sous Linux.

Ligne de commande

Sous Windows : 

xtrkcad [/c config] [/v] [/l logfile] [/d logcontrol] fichier de mise en page

Sous Linux: 

xtrkcad [-c config] [-v] [-l logfile] [-d logcontrol] fichier de mise en page

Paramètres de ligne de commande

c config
configuration à utiliser. Cela vous permet d’avoir plusieurs configurations distinctes (ex : fichiers de paramètres, options d’affichage).

layoutfile
nom du fichier à charger pour la mise en page. Ce paramètre à la priorité sur l'option à reprendre avec le dernier plan (Section 2.10.7).

Paramètres de débogage

Les paramètres suivants sont destinés au débogage et intéressent principalement les développeurs.

v
verbose mode.

d logcontrol
spécifie la quantité de journalisation. L'argument logcontrol a le format logmodule logmodule[=level]. logmodule définit les fonctionnalités à consigner, le niveau facultatif spécifie le niveau de journalisation. Les valeurs par défaut sont qu'aucun module n'est consigné et loglevel est 1 si le paramètre logonctrol est omis.

l logfile
set logfile, nécessaire uniquement avec l'option de débogage.
