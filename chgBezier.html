2.2.8.9 Modifier une Bezier

Comme une Bézier est définie par la position de ses points de contrôle, sa forme ne peut être modifiée qu'en faisant glisser un point de contrôle.

  1. Cliquez sur le bouton Modifier () pour activer la fonction Modifier.

  2. Cliquez sur la voie. Les points de contrôle apparaîtront

  3. Clic gauche d'un point de contrôle et Gauche-glisser pour le déplacer.

  4. Répétez l'opération pour un autre point final si vous le souhaitez

  5. Une fois que vous êtes satisfait, appuyez sur la barre d'espace ou Entr, ou pour annuler, appuyez sur Ech.

Remarque : Le point de fin d'une Bézier ne sera modifiable que s'il n'est pas connecté. Les deux points de contrôle interne peuvent être modifiés même si la voie est connectée.
