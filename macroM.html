2.8 Menu Macro

Le menu macro contient des commandes pouvant être utilisées pour les commandes Enregistrer (Record) et Lecture (Playback).

Enregistrer - affiche la boîte de dialogue du même nom (après avoir demandé un nom de fichier). L'état actuel du plan et la plupart des actions ultérieures seront sauvegardés dans le fichier d'enregistrement.

  • Vous pouvez insérer des messages dans le fichier d'enregistrement en appuyant sur le bouton Message de la boîte de dialogue Enregistrer, puis en entrant un message dans l'affichage du texte.

  • Appuyez sur le bouton Terminer (End) pour mettre fin à la saisie du message et reprendre l'enregistrement.

  • Appuyez sur le bouton Stop pour arrêter l'enregistrement.

Lecture (Playback): affiche la boîte de dialogue Lecture (après avoir demandé un nom de fichier). C'est utilisé pour lire des sessions précédemment enregistrées. Remarque : l'état du plan sera enregistré lors de la lecture, puis restauré.

La commande Enregistrer est utile s'il devient nécessaire de signaler un bogue dans XTrackCAD.
