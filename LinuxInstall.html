1.5.3 Linux Installation

XTrackCAD pour LINUX est contenu dans une archive à extraction automatique (exécutable).

  1. Installation à partir de l'archive auto-extractible.
    Après le téléchargement, ouvrez une ligne de commande puis
    ./xtrkcad-setup-5.2.0GA.i386.sh --prefix=/usr/local --exclude-subdir
    Ça installera l'exécutable dans /usr/local/bin. Un répertoire nommé xtrkcad sera créé dans /usr/local/share et tous les fichiers y seront décompressés.
  2. Installation avec un paquet .rpm sous Fedora (pour mémoire Fedora fait partie de la famille Red Hat) :
    Après le téléchargement, ouvrez une ligne de commande puis
    dnf install /home/[chemin du paquet ...]/xtrkcad-setup-5.2.0GA-1.x86_64.rpm
    Ça installera l'exécutable dans /usr/local/bin/xtrkcad/... Un répertoire nommé xtrkcad sera créé dans /usr/local/share/... et tous les fichiers y seront décompressés.
    La traduction en français du logiciel XtrkCAD sera installée dans usr/local/share/locale/fr_FR/LC_MESSAGES/xtrkcad.mo

Si vous installez XTrackCAD dans un autre répertoire, définissez la variable d'environnement XTRKCADLIB pour qu'elle pointe vers ce répertoire.
