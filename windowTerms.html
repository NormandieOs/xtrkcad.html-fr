B.1 Termes Windows

Lorsque les éléments de menu et de sous-menu sont indiqués par File>Open, celà signifie qu'il vous faut sélectionner le menu File, puis sélectionner ensuite l'élément de menu Open.

Balloon Help (Infobulles) - bref message d'aide qui s'affiche lorsque vous déplacez le pointeur de la souris sur un contrôle.

Canvas (zone de dessin) - zone d'une fenêtre où des objets sont dessinés.

Check Box (case à cocher) - bouton avec un petit indicateur carré à gauche du texte. Une coche apparaît dans l'indicateur lorsque le bouton est sélectionné.

Click (cliquer) - appuyer puis relâcher le bouton gauche de la souris.

Drag (faire glisser) - maintenir enfoncé le bouton gauche de la souris tout en déplaçant la souris pour faire glisser le curseur sur l'écran.

Dialog (dialogue) - type de fenêtre dans laquelle vous entrez des informations.

Drop Down List (Liste déroulante) - une liste déroulante qui ne s'affiche que lorsque vous cliquez sur le bouton "déroulant".

Edit Box (zone de modification) - zone où vous pouvez entrer une valeur sous forme de chaîne, d'entier ou de virgule flottante.

Highlighted (en surbrillance) - partie d'une zone de dessin qui est en vidéo inverse.

Left-Click (Clic gauche) - Identique à Cliquer, utilisé dans les contextes où Clic droit est également utilisé.

Left+Drag (Clic gauche+glisser) - identique à Drag, utilisé dans les contextes où Droite + Drag est également utilisé.

Mouse Pointer (Pointeur de souris) - le plus souvent c'est une flèche que vous pouvez utiliser pour pointer différents objets sur votre écran. Lorsque le pointeur se trouve sur un objet, vous pouvez cliquer sur un bouton de la souris pour exécuter une action sur cet objet (par exemple, sélectionner cet objet).

Press (appuyer sur) - cliquer sur un bouton.

Radio Button (bouton radio) - bouton avec un petit indicateur rond à gauche de son texte. L'indicateur est plein lorsque le bouton est sélectionné

Radio Button Group (menu à options) - liste d'options qui proposent des choix mutuellement exclusifs.

Resizing Border (bordure de redimentionnement) - bordure épaisse autour de fenêtres redimensionnables que vous pouvez utiliser pour modifier la taille de la fenêtre.

Right-Click (clic droit) - appuyez puis relâchez le bouton droit de la souris.

Right Drag (clic droit+glisser) - maintenez le bouton droit de la souris enfoncé et déplacez la souris pour "faire glisser" le curseur à l'écran.

Select (sélectionner) - cliquez sur un objet pour indiquer que vous voulez faire quelque chose avec lui.

Shift+Left-Click (Maj + clic gauche) - Cliquez avec le bouton gauche de la souris en maintenant la touche Maj enfoncée.

Shift+Right-Click (Maj + clic droit) - Cliquez avec le bouton droit de la souris en maintenant la touche Maj enfoncée.
