2.2.14.3 Aligné sur un autre objet

Faire pivoter et aligner un objet avec un autre objet.

Un menu contextuel contenant des éléments de menu supplémentaires vous permet d'Aligner les objets sélectionnés sur tout autre objet de la présentation. La commande Aligner est exécutée comme suit :

  1. Sélectionnez le ou les objets à faire pivoter.

  2. Cliquez sur le bouton Rotation () pour activer la fonction Pivoter.

  3. Le pointeur de la souris survolant le point autour duquel vous souhaitez faire pivoter le ou les objets, Maj+clic droit pour afficher le menu Rotation, puis sélectionnez Aligner.

  4. Cliquez avec le bouton gauche de la souris sur l'objet (sélectionné) à faire pivoter.

  5. Cliquez avec le bouton gauche sur l'objet avec lequel vous souhaitez aligner l'objet sélectionné. L'objet sélectionné pivotera pour s'aligner sur l'objet de référence.

La rotation peut également être obtenue en utilisant l'une de ces méthodes alternatives :

Gauche+glisser en utilisant la souris. Voir Section 2.2.14.1 pour plus de détails.

Saisir le nombre de degrés de rotation prédéfinis ou spécifiés à l'aide du menu Clic droit. Voir Section 2.2.14.2 pour plus de détails.
