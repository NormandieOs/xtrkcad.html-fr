1.9.1 Survol des répertoires

XTrackCAD utilise deux répertoires sur votre système :

  • Le répertoire d'installation de XTrackCAD Install, qui contient les fichiers de données et de paramètres utilisés par XTrackCAD.

  • Le répertoire de travail XTrackCAD Working, qui contient les fichiers de configuration et de contrôle.

Sous MS Windows, le répertoire Install est défini lors de l'installation d'XTrackCAD. Tous les fichiers de données (paramètres, démos, exemples) se trouvent dans le sous-répertoire Install : C:\Program Files (x86)\XTrkCAD 5.1.2a\share\xtrkcad. Le répertoire Working est dans votre profil personnel : C:\Users\"login"\AppData\Roaming\XTrackCad

Sous Linux, le répertoire Install est généralement /usr/local/lib/xtrkcad/, mais vous pouvez le modifier en définissant la variable d'environnement XTRKCADLIB avant de démarrer XTrackCAD. Le répertoire de travail est {HOME}/.xtrkcad/.

Sous Mac OSX, le répertoire Install est le dossier d'application, généralement situé dans le répertoire /Applications. Tous les paramètres, démos et exemples inclus se trouvent à l'intérieur du paquet dans le répertoire /Applications/xtrkcad/Contents/Resources/xtrkcad. Le répertoire Working est créé comme .xtrkcad - un répertoire caché - sous le répertoire de l'utilisateur actif dans /Users Traduit avec www.DeepL.com/Translator (version gratuite)

Les fichiers de mise en page (*.xtc) et les fichiers d'exportation (Section 2.5.1) (*.xti) peuvent être enregistrés où vous le souhaitez, comme indiqué dans les boîtes de dialogue Sauver sous (Save File) et Ouvrir (Open File).

Dans MacOS Catalina, le programme n'a pas accès à certains répertoires - les répertoires Documents et Download en particulier.

Vous trouverez des exemples de modèles dans le répertoire examples situé dans le répertoire d'installation XTrackCAD (Pour Windows : C:\Program Files (x86)\XTrkCAD 5.1.2a\share\xtrkcad\examples).
