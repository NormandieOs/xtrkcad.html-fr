2.4 Menu Editer (Edit)

Le Edit Menu propose des commandes qui affectent le plan principal (section 3.1).

Au-dessus (Above) - Déplace l'objet sélectionné au premier plan.

Au-dessous (Below) - Déplace l'objet sélectionné en arrière-plan.

Copier (Copy) - Copie les objets sélectionnés dans le presse-papiers.

Couper (Cut) - Déplace les objets sélectionnés dans le presse-papiers.

Effacer (Delete) - Supprime les objets sélectionnés.

Tout désélectionner (Deselect All) - désélectionne tous les objets du plan. La même action peut être obtenue en appuyant sur la touche ESC.

Inverser la sélection (Invert Selection) - sélectionne tous les objets non sélectionnés et désélectionne tous les objets précédemment sélectionnés.

Déplacer vers le calque actuel (Move To Current Layer) - déplace tous les objets sélectionnés vers le calque actuel (Section 2.9.5).

Coller (Paste) - Copie le contenu du presse-papiers sur le plan. Les nouvelles pièces collées sont dans l'état Sélectionner et peuvent être facilement déplacées. Tous les objets précédemment sélectionnés sont désélectionnés.

Refaire (Redo) - Annule la dernière commande d'annulation. La commande Refaire peut également être appelée à l'aide du bouton Refaire.(Section 2.4.5).

Tout sélectionner (Select All) - Sélectionne tous les objets du plan.

Sélectionner le calque actuel (Select Current Layer) - sélectionne tous les objets du plan actuel (Section 2.9.5).

Sélectionner une voie isolée (Select Stranded Track) - sélectionne tous les segments de voie qui ne sont connectés à aucun autre objet. Cela aide à nettoyer un dessin après de nombreuses modifications.

Voies fines, moyennes et épaisses (Thin, Medium and Thick Tracks) - définissez la largeur de trait des rails des voies sélectionnées.

Tunnel (Tunnel) - Masque ou affiche les voies sélectionnées.

Annuler (Undo) - Annule la dernière commande. Les dix dernières commandes peuvent être annulées. Si aucune commande ne peut être annulée, l'élément Annuler est désactivé. La commande Annuler peut également être appelée à l'aide du bouton Annuler. (Section 2.4.5).

Pour plus d'informations sur l'utilisation des objets sélectionnés, voir la commande Select (Section 2.2.15).
