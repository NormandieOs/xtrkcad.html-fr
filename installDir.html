1.9.2 Répertoire d'installation

  • xtrkcad.xtq est le fichier de paramètres principal qui contient les définitions de chaque échelle. Pour les démos il inclut également les définitions d'aiguillages et les fichiers de lecture (.xtr).

  • Le répertoire params contient les fichiers *.xtp qui sont les fichiers de paramètres de la bibliothèque XTrackCAD. Ces fichiers (atlasn.xtp, pecon55.xtp, ...) contiennent des définitions pour les aiguillages, les éléments de voie et les structures. On peut trouver ces fichiers dans la boîte de dialogue Fichiers de Parametres (Section 2.5.5).

  • xtrkcad.chm (sous MS-Windows) est le fichier d'aide de XTrackCAD.

  • xtrkcad.exe (MS-Windows) ou xtrkcad (Linux) est le programme XTrackCAD .

  • xtrkcad.tip contient les informations de Conseil du jour accessibles à partir du menu Aide ou affichées au démarrage.

  • xtrkcad0.ini spécifie l'emplacement du répertoire de travail de XTrackCAD (MS-Windows uniquement).

  • Le répertoire demos contient les *.xtr, qui sont les fichiers de démonstration de XTrackCAD pouvant être exécutés via l'élément de menu Aide>Demos .

  • Le répertoire examples contient des exemples de fichiers .xtc
