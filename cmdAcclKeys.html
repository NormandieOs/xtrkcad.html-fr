3.4 Actions de la souris

XTrackCAD utilise les boutons gauche et droit de la souris pour manipuler les voies et autres objets sur le tracé. Leur signification dépend de la commande particulière :

Commande sélectionner

Les actions de la commande Sélectionner sont affectées par le mode sélectionné dans Options->Commande->Mode sélection. Remarque - Les objets d'un module de calque sont sélectionnés et désélectionnés en tant que groupe.

Clic-gauche
Mode Elément unique -> Sélectionne un objet à la fois, désélectionne les autres objets. Mode ajout d'un objet -> Ajoute les objets non sélectionnés, désélectionne si l'objet est déjà sélectionné. Si le clic se fait sur une zone sans objets et que Désélectionner tout est défini, tous les objets sont désélectionnés.
Maj+Clic-gauche
Sélectionne toutes les voies connectées depuis celle sur laquelle on a cliqué jusqu'à l'élément sélectionné suivant dans chaque direction.
Ctrl+Clic-gauche
Mode Elément unique -> Ajoute un objet à la sélection, désélectionne si l'objet est déjà sélectionné. Mode Ajout d'un objet -> Sélectionne uniquement cet objet et désélectionne les autres objets. Si le clic se fait sur une zone sans objets et que Désélectionner tout est coché, tous les objets sont désélectionnés.
Double-Clic-gauche
Ouvre Modifier une Cornu et dessiner des objets autres que du texte, active le lien et les notes de document, sinon ouvre Décrire.
Gauche+glisser
Mode Elément unique -> Sélectionne tous les objets dans la zone élastique et désélectionne tous les autres. Ajouter à la sélection -> Ajoute des éléments dans la zone élastique.
Maj+Gauche+glisser
Déplace les objets sélectionnés s'ils se trouvent dans la zone sélectionnée, si les objets sélectionnés sont désactivés, comme le Gauche+glisser.
Ctrl+Gauche+glisser
Fait pivoter les objets sélectionnés s'ils se trouvent dans une zone sélectionnée, si les objets sélectionnés sont désactivés, comme le Gauche+glisser.
Clic-droit
Affiche un menu contextuel de commandes (dans les commandes Sélectionner, Déplacer, Modifier et Faire pivoter). Si une voie est sélectionnée, le menu comporte des opérations à effectuer, sinon, il comporte un menu contextuel de commandes.
Maj+Clic-droit
Affiche un menu contextuel de commandes. Affiche un menu contextuel autre que celui du clic droit.
Droit+glisser
Désélectionne tous les objets dans la zone élastique de l'ensemble actuel.
Molette de souris
Effectue un zoom avant et arrière sur la zone de dessin. Les ordinateurs modernes équipés de trackpads associent souvent ce geste à un glissement à deux doigts.

Lors du déplacement et de la rotation ( Placer les aiguillages et les structures, Déplacer les grilles d'impression et d'accrochage, ... ) :

(Maj+Gauche+glisser)
Déplace les objets ou la grille.
(Ctrl+Gauche+glisser)
Fait pivoter les objets ou la grille autour du point de pivot, qui est le point où vous avez commencé l'opération de glissement.

Certaines autres commandes utilisent la touche Maj ou Ctrl pour modifier la signification des pressions sur les boutons.
