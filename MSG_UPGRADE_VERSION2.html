C.2.39 La version du fichier %ld est supérieure à celle supportée ...

Le fichier du projet que vous essayez de charger a été créé à l'aide d'une version plus récente d'XTrackCAD. Pour charger ce fichier, vous devez mettre à niveau votre version d'XTrackCAD vers au moins la version identifiée dans le message d'erreur.

Tirez pleinement parti des dernières fonctionnalités d'XTrackCAD en visitant le site Web sourceforge d'XTrackCAD et en téléchargeant une version appropriée.
