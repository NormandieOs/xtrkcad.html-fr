C.2.70 Veuillez sélectionner une imprimante dans la boite de dialogue de configuration d'impression.

Les marges d'impression sont spécifiques à l'imprimante sélectionnée, mais aucune imprimante sélectionnée n'est choisie. Vous pouvez sélectionner une imprimante dans la boite de dialogue Configuration d'impression.
