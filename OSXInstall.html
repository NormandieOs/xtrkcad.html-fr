1.5.2 OSX Installation Mac Installation

XTrackCAD pour OSX est livré sous forme de package contenant une application.

  1. Démarrez l'installation en ouvrant le package et en faisant glisser l'application contenue dans le dossier Applications à l'aide du raccourci dans le package. L'application contient le programme, la documentation, les fichiers de paramètres et d'exemples.

  2. Pour fonctionner sur Apple MacOS, le package XQuartz doit d'abord avoir été installé - voir http://www.xquartz.org. XQuartz devra être mis à niveau après chaque mise à niveau de la version du système d'exploitation.

  3. Sécurité d'exécution OSX Lors d'une première exécution, l'application devra être autorisée par OSX. Une invite OSX vous demandera si elle doit être ouverte. Puis, elle ne pourra toujours pas être exécutée. Après l'avoir ouverte pour la première fois, allez dans System Preferences>Security & Privacy et vous verrez que l'application est nommée comme ayant été bloquée. Appuyez sur le bouton Run Anyway et le programme devrait démarrer. Cette séquence peut être nécessaire après chaque mise à niveau du système d'exploitation.
  4. Sur Mac OS Catalina, le programme devra être lancé en cliquant avec le bouton droit de la souris et en sélectionnant Open.
