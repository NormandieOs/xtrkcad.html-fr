2.10.2 Fenêtre options de commande

La boite de dialogue Options de Commande est appelée depuis le menu Options (Section 2.10)

La boite de dialogue Options de Commande contient des éléments qui contrôlent le comportement de certaines commandes.

Commande par défaut - sélectionne Propriétés (Section 2.2.13) ou Sélectionner (Section 2.2. 16) comme commande par défaut qui est appelée lorsque toute autre commande (sinon Maintenue (Section 2.10.8)) est annulée ou complétée.

Clic droit - affichera soit un menu contextuel des commandes disponibles, soit des options spécifiques à la commande selon que la touche Maj est maintenue enfoncée ou non. Cette option vous permet de choisir l'opération qui se produit.

Mode sélection - soit (a) sélectionnera uniquement l'élément courant, sauf si la touche Ctrl est actionnée, soit (b) ajoutera l'élément courant à la sélection, sauf si la touche Ctrl est actionnée, auquel cas il sélectionnera uniquement cet élément.

Zéro-sélection - si elle est sélectionnée, effectuera Désélectionner tout si la sélection n'est pas sur un objet.
