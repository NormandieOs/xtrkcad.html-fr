2.1.11 Bloc

Pour créer un bloc, sélectionnez d'abord les segments de voie du bloc (Modifier/Sélectionner ou Ctrl+E) , puis sélectionnez Créer un bloc dans le menu "Ajouter/Bloc" (Add/Block) Section 3.3.1 ou avec le bouton bloc dans la hot-bar. Un bloc comprend un nom et un script. Le nom sert uniquement à l'identification et le script est utilisé par le logiciel de contrôle du plan. Le script doit fournir toutes les informations nécessaires au logiciel de contrôle du plan pour déterminer l'occupation du bloc. Il peut s'agir d'un extrait de code permettant de récupérer l'état de l'occupation du bloc ou de l'adresse du capteur, etc.

La boîte de dialogue crée un bloc qui contient des espaces pour deux chaînes de texte. L'un est le nom et l'autre est un script d'occupation de bloc. Le nom identifie le bloc et le script est une information utilisée par le logiciel de contrôle du plan pour détecter l'occupation du bloc. Le script peut être un extrait de code ou des informations d'adresse de périphérique d'E/S, etc.
