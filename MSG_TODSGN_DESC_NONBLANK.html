C.2.4 Tous les champs de description d'un aiguillage ...

Les descriptions d'aiguillages doivent être aussi précises que possible. Tous les champs de description répertoriés dans Conception d'aiguillage (Turnout Designer) doivent contenir les informations appropriées.

Référez vous à la section 2.9.9 pour plus de détails.
