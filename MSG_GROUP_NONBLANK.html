C.2.5 Tous les champs de la boîte de dialogue "Grouper" doivent contenir des données ...

La boîte de dialogue Grouper contient le fabricant, la description et le numéro de série (peut-être dupliqué dans le titre) de la nouvelle définition. Les valeurs initiales sont basées sur les aiguillages et les structures sélectionnés ou sur le dernier objet Dissocié (Ungroup). La boîte de dialogue contient également une case à cocher (Remplacer par un nouveau groupe ?), à utiliser si vous souhaitez remplacer les objets sélectionnés par une nouvelle définition. Tous les champs répertoriés dans cette boîte de dialogue doivent être remplis.

Référez vous à la section 2.9.4 pour plus de détails.
