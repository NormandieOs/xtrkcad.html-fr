2.2.13 Monter/descendre les Elévations

Cette commande vous permet d'élever ou d'abaisser l'élévation de toutes les voies sélectionnées.

La valeur entrée est ajoutée à toutes les élévations définies sur les voies sélectionnées. Une valeur négative fera baisser les élévations.
