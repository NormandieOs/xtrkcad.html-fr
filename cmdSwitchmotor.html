2.1.11 Moteur d'aiguillage

Un moteur d'aiguillage est créé en sélectionnant Moteur d'aiguillage dans le menu Ajouter Section 3.3.1 ou sur l'icône de la barre de raccourci, puis en cliquant sur un aiguillage du réseau. Un moteur d'aiguillage reçoit un nom et trois scripts. Le nom n'est utilisé que pour l'identification et les trois scripts sont utilisés par le logiciel de contrôle du réseau. Les trois scripts sont destinés à placer l'aiguillage dans sa position «normale», à inverser la logique de commande et à créer un script pour lire un capteur de position. Ces scripts peuvent être des extraits de code ou des adresses de périphériques d'E/S, etc.

La boîte de dialogue Créer un moteur d'aiguillage (Create switch motor) comporte des espaces pour quatre chaînes de texte. L'un est le nom et les trois autres sont des scripts pour Normal, Inversé et Sens des aiguilles. Le nom identifie le commutateur et les scripts sont des informations utilisées par le logiciel de contrôle de la configuration pour mettre l'aiguillage dans sa position normale, inverser sa logique de commande et détecter la position des aiguilles. Les scripts peuvent être des extraits de code ou des informations d'adresse de périphérique d'E/S, etc.
