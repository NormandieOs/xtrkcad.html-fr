2.2.9 Déplacement des étiquettes

Cette commande permet de déplacer les étiquettes (Track Description) d'élévation, pente, gare (Section 2.2.4), voie courbe (Section 2.1.2), hélicoïde (section 2.1.4) ou aiguillage(section 2.7).

La commande Déplacer l'étiquette peut être appelée à l'aide de l'élément de menu Modifier>Déplacer l'étiquette (Change>Move Description), du bouton Déplacer l'étiquette ou du menu contextuel affiché par un Clic droit souris sur le plan principal.

Cliquez sur le bouton Déplacer l'étiquette pour activer la commande Déplacer l'étiquette. Utilisez Gauche+Glisser pour déplacer l'étiquette vers un nouvel emplacement. Une fois déplacée, l'étiquette devient fixe dans une position relative à l'objet. Les déplacements ultérieurs de cet objet entraîneront le déplacement de l'étiquette.

Cette opération peut également être appelée par Ctrl+Gauche+Glisser en mode Sélection (Section 2.2.15).

Maj+clic droit (Section 3.4) peut être utilisé pour basculer l'affichage de voies courbes, voies de Bézier, cornu, hélicoïdes ou Étiquettes d'aiguillages.
