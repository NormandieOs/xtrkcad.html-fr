C.2.45 Description des étiquettes non visibles

La commande Déplacer la description (Move Description) ne peut être utilisée que lorsque les libellés sont visibles sur le plan. Les étiquettes ne sont pas visibles lorsque l'échelle des étiquettes (Label Scale) est définie par une valeur inférieure à l'échelle actuelle du dessin ou si aucun élément n'est activé dans la section Etiquettes activées (Label Enable) d'Options d'affichage (Display Options).

Vérifiez la boîte de dialogue Options d'affichage (Display Options) (Options>Afficher (Display)) pour déterminer si :

  • Echelle d'étiquettes (Label Scale) est définie par une valeur inférieure ou égale à l'échelle actuelle du dessin.
  • Au moins un attribut de la section Etiquettes activées (Label Enable) est sélectionné. Assurez vous que le type d'étiquette que vous souhaitez déplacer est sélectionné.

Reportez-vous à la section 2.2.9 pour plus de détails sur les Déplacer l'étiquette (Move Description) ou à la section 2.10.3 pour plus de détails sur les attributs Options>Affichage (Display).
