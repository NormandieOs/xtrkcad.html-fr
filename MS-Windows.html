2.5.7.2 Microsoft Windows

La boîte de dialogue Windows Réglages d'impression vous permet de configurer l'utilisation de l'imprimante par l'application.

La fenêtre Configuration de l'imprimante contient:

Nom : liste déroulante permettant de sélectionner une imprimante particulière. Les entrées de cette liste sont définies par Windows.

Etat - état actuel de l'imprimante sélectionnée.

Type - le type et le modèle d'imprimante sélectionné.

Emplacement - Emplacement de l'imprimante s'il est défini dans les propriétés de l'imprimante.

Commentaire - Commentaire défini dans les propriétés de l'imprimante.

Papier Taille - liste de divers formats de papier nord-américains et européens.

Papier Source - liste des différents magasins à papier associés à l'imprimante sélectionnée.

Orientation - un bouton radio pour sélectionner l'orientation Paysage ou Portrait.
