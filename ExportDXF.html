2.5.3 Exporter en DXF

Cet élément de menu affiche une boîte de dialogue de sélection de fichier dans laquelle un nom de fichier pour les objets exportés peut être saisi. Tous les objets sélectionnés (Section 2.2.15) sont exportés dans le fichier. Le fichier d'exportation aura une extension .dxf.

Les fichiers exportés peuvent ensuite être importés dans d'autres programmes de CAO.
