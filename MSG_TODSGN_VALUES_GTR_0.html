C.2.6 Toutes les valeurs spécifiées lors d'une "conception d'aiguillage" doivent être ...

Les dimensions de l'aiguillage doivent être aussi précises que possible. Toutes les valeurs spécifiées dans Conception d'aiguillage (Turnout Designer) doivent être supérieures à 0.

Référez vous à la section 2.9.9 pour plus de détails.
