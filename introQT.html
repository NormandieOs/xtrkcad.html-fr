1.7 Prise en main rapide

Quand XTrackCAD est lancé, il affiche la vue principale (Section 3.1) et la carte (Section 3.2.2). La carte est utilisée pour selectionner la partie à afficher dans la vue principale quand vous ne souhaitez pas tout afficher en même temps.

La barre de menu en haut de la fenêtre principale contient les menus standard Windows. Certains seront particulièrement utiles aux nouveaux utilisateurs :

  • Le menu Fichier>Quitter (File>Exit) permet de quitter XTrackCAD. Vous pouvez aussi sortir en utilisant le menu System dans le coin en haut à gauche de la fenêtre.

  • Le menu Fichier>Ouvrir (File>Open) permet d'ouvrir des dessins existants. A l'installation, le répertoire par défaut contient quelques exemples que vous pouvez parcourir.

  • Le menu Aide (Help) peut être utilisé pour lire le fichier d'aide d'XTrackCAD. De plus, le menu Aide>Demos (Help>Demos) propose un bon nombre de demonstrations qui illustrent les fonctions variées d'XTrackCAD

  • Le menu Options>Plan (Options>Layout) indique l'échelle et la dimenssion du dessin.

  • Le menu Options>Preferences propose quelques paramètres moins souvent modifiés. Vous pouvez utiliser ce menu pour choisir le système métrique (Section 2.10.7).
