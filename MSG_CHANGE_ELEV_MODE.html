C.2.20 Impossible de modifier le mode élévation.

Dans la commande Profil (Profile), le mode Elevation de ce bout de voie ne peut être modifié que s'il est Défini (Defined), Ignoré (Ignored) ou Aucun (None). Auquel cas, aprés avoir cliqué sur un bout de voie ce qui fait apparaitre un gros point rouge, ce dernier deviendra modifiable par la commande élévation.

Référez vous à la section 2.2.5 pour plus de détail.
