A.2 Objets manquants dans la Hot Bar !

Après l'installation de XTrackCAD, la Barre rapide peut ne pas être active; les flèches à gauche et à droite de la barre de défilement sont grises au lieu de noires, la barre de défilement est vide et ne fait pas défiler l'écran.

Il n'y a pas de mal, il vous suffit de charger une voie ou une structure de fichier de paramètres. Ouvrez la fenêtre Paramètre (Fichier>Paramètre), puis cliquez sur Parcourir. Une liste de fichiers *.Xtp portant des noms tels que atlasn55.xtp doit apparaître. Sélectionnez le ou les fichiers qui conviennent à vos besoins, puis cliquez sur le bouton OK. Cliquez sur OK dans la boîte de dialogue Fichier de paramètres et vous devriez maintenant avoir les objets affichés sur la barre de raccourcis.

Notes:

  1. Seuls les éléments qui correspondent à l'échelle sélectionnée sont disponibles dans la hot bar. Par conséquent, si vous êtes certain que le fichier de paramètres a été chargé via la boîte de dialogue Fichier>Paramètre , assurez-vous que la bonne échelle a été sélectionnée pour la présentation (Options>Présentation (Layout) ).

  2. Si une longue liste de fichiers de paramètres est sélectionnée, le mouvement dans la barre de raccourcis peut être lent. Si vous cliquez avec le bouton droit sur la barre de raccourcis, une liste des jeux de paramètres disponibles s'affiche. Avec un Clic gauche dans le jeu de paramètres souhaité, la barre de défilement passe immédiatement au premier élément de cet ensemble de paramètres.

  3. Lors de la création d'un nouveau plan ou lorsque XTrackCAD est démarré pour la première fois, l'échelle du plan est définie par défaut sur l'échelle N. N'oubliez pas de changer l'échelle pour voir les voies d'autres échelles dans la Barre rapide !
