D.1 Version 5.x

La version 5.x.x de XTrackCAD a introduit de nombreuses améliorations et ajouts, notamment des objets de contrôle du plan, des voies de Cornu et des voies et lignes de Bézier. Elle est compatible avec les versions antérieures mais si les nouveaux objets sont inclus dans les plans, les fichiers ne seront pas lisibles par les versions antérieures.

Les utilisateurs doivent faire une sauvegarde de leurs fichiers de plans s'ils veulent utiliser le niveau de code précédent avant de les ouvrir.
