2.9.5 Menu Calques (Layers Dialog)

Chaque objet est contenu dans un calque. Les objets créés indépendamment d'autres objets (tels que les voies droites (Section 2.1.6) ou du texte (Section 2.3.7)) sont créés dans le calque courant. Les objets associés à d'autres objets (telles que les voies créées par Modifier (Section 2.2.7) ou Raccorder (Section 2.2.6)) sont créées dans le même calque que le ou les objets associés.

La boîte de dialogue Calques (Layers) vous permet de modifier divers attributs de chacun des 99 calques possibles.

Les attributs associés à un calque sont stockés dans le fichier calque associé et s'appliquent uniquement au calque ouvert au moment d'une modification. Il est possible d'appliquer des attributs d'un fichier de calque à un autre à l'aide des boutons Charger (Load) et Sauver (Save).

La liste déroulante située en haut de la boîte de dialogue sélectionne le calque qui vous intéresse. Les N° de calques non vides sont suivis d'un +.

La zone d'édition Nom (Name) contient le nom du Calque (Layer). Ce nom figure dans la liste déroulante Calque courant (Current Layer) de la barre d'outils (Tool-bar) (Section 3.1). Le Nom (Name) est également affiché comme bulle d'aide pour le Bouton de calque (Layer Button) correspondant.

Le bouton Couleur permet de spécifier la couleur des voies si le bouton radio couleur de calque (Color Layers) (Section 2.10.3) de la boîte de dialogue Afficher (Display) est coché.

La case à cocher Visible détermine si le Calque (Layer) est affiché dans la fenêtre principale (Section 3.1).

La case à cocher Gelé (Frozen) indique que les voies de ce calque (Layer) ne peuvent ni être modifiées ni supprimées.

La case à cocher Sur la carte (On map) indique si le Calque (Layer) est affiché sur la fenêtre de navigation (Section 3.2.2).

Vous ne pouvez pas décocher Visible ou cocher Gelé (Frozen) pour trouver le Calque en cours (Current Layer).

Le champ Qté affiche le nombre d'objets du calque sélectionné.

Le bouton Sauver enregistre les attributs du calque dans le fichier xtrkcad.ini afin qu'ils puissent être chargés dans un fichier existant ou bien un nouveau. Une fois sauvegardées et sauf réinitialisation d'XTrackCAD , par défaut, les informations des calques modifiés sont appliquées à chaque nouveau fichier.

Le bouton Charger (Load) lit les attributs des calques à partir du fichier xtrkcad.ini, puis les applique à chaque calque du fichier du projet actuel.

Le bouton Défauts (Defaults) permet de restaurer les attributs par défaut des paramètres XTrackCAD.

La partie inférieure de la boîte de dialogue contient la zone d'édition du Nombre de boutons de calques (Number of Layer Buttons). C'est pour contrôler le nombre de boutons de calque affichés dans la barre d'outils (section 3.1). Ces boutons facilitent le basculement entre les calques. Le nombre maximal de boutons de calque pouvant être affichés est de 20. Par conséquent, vous devez utiliser les calques 1 à 20 pour les objets que vous allez afficher et masquer fréquemment.

Les boutons de calque de la barre d'outils (Section 3.1) contrôlent le calque correspondant au calque actuel ainsi que les Calques affichés sur le plan principal. Chaqu'un de ces boutons est un raccourci de la case à cocher Visible de la boîte de dialogue Calques (Layers) du calque correspondant.

Le calque courant est sélectionné en choisissant une des valeurs de la liste déroulante des calques. Si le calque que vous avez sélectionné n'était pas visible, il pourra l'être.

Vous pouvez masquer (ou afficher, s'ils ne sont pas visibles) les différents calques en appuyant sur l'un des boutons Calques (Layer Buttons) de la barre d'outils (si vous avez configuré la barre d'outils (Section 2.12.2) pour afficher les boutons de calques). Si un calque n'est pas visible, le bouton du calque correspondant pourra être activé. Un calque actif ou l'actuel ne peut pas être masqué.

La couleur de chaque bouton de calque indique la couleur qui sera utilisée pour dessiner les voies de ce calque, si vous sélectionnez Couleur des calques (Color Layers) (Section 2.10.3) dans la boîte de dialogue Options>Affichage....

Tous les objets sélectionnés sont désélectionnés lorsque le calque est défini comme non visible. Si un calque n'est pas visible, vous ne pouvez pas sélectionner d'objets dans ce calque. C'est comme si les objets n'existaient pas jusqu'à ce que vous le montriez à nouveau.

La commande Déplacer vers le calque actuel (Move To Current Layer) du menu Edition (section 2.4) déplacera tous les éléments sélectionnés (Section 2.2.15) sur le calque actuel.

Les objets sont dessinés dans la fenêtre de la carte, si Sur la carte (On Map) est coché pour le calque sur lequel ils se trouvent.
