C.1 Types de message

XTrackCAD génère trois types de messages :

Les avis apparaissent dans leur propre fenêtre et vous obligent à appuyer sur un bouton pour faire un choix ou les ignorer.

Les messages d'erreur, qui apparaissent dans la barre d'état (Section 3.1) et qui sont accompagnés d'un bip.

Les messages d'information, qui apparaissent dans la barre d'état ( Section 3.1 ) sans bip.

Les avis et les messages d'erreur pouvant être rencontrés sont répertoriés ici avec une explication. L'élément de menu Messages récents de l'Aide ( Section 2.6 ) contient la liste des messages les plus récents. La sélection d'une des entrées présente des informations sur un message particulier.

La notation abc|xyz indique que le message peut apparaître avec l'une ou l'autre des valeurs (abc ou xyz). Les éléments d'MAJUSCULE indiquent des valeurs numériques (telles que des longueurs) ou des chaînes (telles que des noms de fichiers).

Section C.2 fournit une explication détaillée de chaque information ou message d'erreur.
