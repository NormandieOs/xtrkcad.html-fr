2.5.1 Exporter (Export)

Cet élément de menu affiche une boîte de dialogue de sélection de fichier dans laquelle on peut saisir un nom de fichier pour les objets exportés . Tous les objets (Section 2.2.15) sélectionnés sont exportés dans le fichier. Le fichier d'exportation aura une extension .xti.

A l'aide de la commande Importer (section 2.5.4), un fichier exporté peut être importé dans un autre projet de plan.
