2.1.12 Signal

Un signal est créé en sélectionnant Signal dans le menu Ajouter Section 3.3.1 ou dans la barre de raccourci. Utilisez la souris pour sélectionner un emplacement (bouton gauche), puis faites glisser (le bouton gauche enfoncé) le signal pour définir son orientation. Une fois le bouton gauche relâché, une boîte de dialogue Créer/Modifier un signal s'affiche et vous permet de renseigner ses propriétés : nom, réglage précis de l'emplacement et de l'orientation, nombre de têtes et aspects.

La boîte de dialogue Créer/Modifier un signal contient des espaces pour le nom du signal, son emplacement et son orientation, le nombre de têtes (1, 2 ou 3) et une liste d'états. Chaque état a un nom et un script. Ce nom peut être le nom de règles (Clear, Aproach, Stop, etc.) ou la ou les couleurs réelles affichées (vert, jaune, rouge, etc.). Le script ne représente que quelques informations permettant au logiciel de contrôle du plan d'affecter l'affichage de l'état - il peut s'agir de l'identificateur d'événement LCC permettant de déclencher l'état ou un extrait de code qui provoque son affichage.
