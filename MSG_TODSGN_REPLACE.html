C.2.36 Ce nom de définition est déjà utilisé. Sauvegardez le ...

La définition que vous essayez de créer utilise le même nom qu'une définition déjà existante. Si vous enregistrez la définition avec le même nom, XTrackCAD écrasera la définition existante en la rendant inaccessible. Si la définition précédente doit être préservée, choisissez un autre nom pour la nouvelle définition. Si nécessaire, la boîte de dialogue Gestion personnalisée (Custom Management) peut être utilisée pour modifier les noms de définition.

Référez vous à la section 2.9.2 pour plus de détail.
