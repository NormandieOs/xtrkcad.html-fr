1.8 Problèmes de performance

Cette section répertorie ce qui peut améliorer les performances d'XTrackCAD.

  • Dans le dialogue Afficher (Section 3.3.11), désactivez Afficher la grille mgnétique ou Modifier la grille ... pour avoir moins de points de grille.

  • Pour accélérer l'initialisation, supprimez les fichiers de paramètres non utilisés (Voir la boite de dialogue Fichiers de Paramètres ... - Section 2.5.5).

  • Fermez la fenêtre Carte de navigation si vous n'en avez pas besoin pour vous déplacer dans le plan. Le menu Fenêtre (Section 2.13) peut être utilisé pour rouvrir la fenêtre Carte (ou Ctrl+Espace).

  • Sur la boite de dialogue Options/Options d'affichage (Section 2.10. 3), désactivez le dessin bouts de voie, diminuez l'échelle 2 rails, augmentez l'échelle Préférences/Pas min. de la grille, réglez Dessin des traverses sur Aucun et désactivez Options d'affichage/Carte en direct.
