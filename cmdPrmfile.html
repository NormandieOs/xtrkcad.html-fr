2.5.5 Fichiers de définitions d'objets (Parameter Files)

Les fichiers de paramètres contiennent des définitions d'aiguillages, de sections de voies et de structures. Les fichiers de paramètres sont chargés et déchargés à l'aide de la boîte de dialogue Parameter Files.

La boîte de dialogue contient une liste de fichiers de paramètres. Chaque fichier est précédé d'une case à cocher. Si la case est cochée, le fichier de paramètres est chargé. Si la case n'est pas cochée, le fichier est déchargé.

Les fichiers de définitions chargés sont filtrés par le paramètre d'échelle en cours avant d'apparaitre dans la Barre d'objets (Hot Bar) ( Section 2.7 ). Si vous désélectionnez un fichier, les définitions sont supprimées de la liste de sélection et de la Barre d'objets. Un fichier déchargé peut être rechargé en sélectionnant le bouton Recharger (le bouton Désélectionner devient Recharger lorsqu'un fichier sélectionné est désélectionné).

Il est possible de changer l'état de plusieurs fichiers à la fois. Le bouton Tout sélectionner vous permet de sélectionner tous les fichiers de paramètres pour l'opération suivante.

Les fichiers déchargés sont supprimés de la liste lorsqu'on quitte XTrackCAD.

Les fichiers peuvent être répertoriés soit par leur nom soit par la description de leur contenu. Cette option est contrôlée par la case à cocher Montrer les noms de fichiers (Show File Names).

Le bouton Parcourir (Browse) vous permet de charger d'autres fichiers de paramètres fournis par d'autres utilisateurs ou téléchargés depuis Internet.

Lors de la première installation, tous les fichiers de paramètres disponibles sont chargés. Vous pouvez décharger des fichiers de paramètres que vous n'utilisez pas pour accélérer le démarrage de XTrackCAD.

L'ordre de cette liste contrôle impacte l'ordre des aiguillages et des structures sur la Barre de sélection d'objets (Hot Bar) (section 2.7). Vous pouvez déplacer un fichier de paramètres en fin de liste en le Désélectionnant, puis en le rechargeant via le bouton Parcourir (Browse).
