3.3.2 Commandes "Change" (Change Commands)

Commandes "Change"

Connecter deux sections de voies - Ctrl+Shift+J

Boite de dialogue Elévation - Ctrl+Shift+E

Raccorder deux voies - Ctrl+J

Boite de dialogue "Properties" - Ctrl+?

Modifier ou prolonger la voie - Ctrl+M

Déplacer
  • description - Ctrl+Shift+Z
  • Objet sélectionné - Ctrl+Shift+M

Fenêtre Profile - Ctrl+Shift+F

Faire pivoter l'objet sélectionné - Ctrl+Shift+R

Sélectionner un outil - Ctrl+E

Couper une voie flexible - Ctrl+Shift+S
