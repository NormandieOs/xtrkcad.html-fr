1.5.1 Microsoft Windows Installation

XTrackCAD est contenu dans un programme d'installation automatique (executable).

  1. Avec Windows Explorer, positionnez vous dans le répertoire dans lequel vous avez chargé ou copié votre nouvelle version de XTrackCAD.

  2. Lancez l'installation en double cliquant sur l'icone du fichier xtrkcad-setup-5.1.2a.exe.

  3. Suivez les instructions du programme d'installation.

  4. L'installation vous permet de définire le répertoire dans lequel XTrackCAD est installé. Le répertoire est crée automatiquement s'il n'existe pas.

  5. Un répertoire nommé XTrkCad 5.1.2a sera crée pendant l'installation. Ce répertoire contient le programme, la documentation, les parametres et les fichiers d'exemple.
