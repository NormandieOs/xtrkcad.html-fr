3.3.5 Commandes de fichier (File Commands)

Commandes de fichier

Créer un nouveau plan - Ctrl+N

Editer une note de description du plan (Section 2.3.3) - Ctrl+Alt+T

Export
  • Objets sélectionnés dans un fichier .xti - Ctrl+Alt+X
  • En Bitmap (Section 2.5.2) - Ctrl+Shift+Q

Importer depuis un fichier .xti - Ctrl+Alt+I

Manipuler des fichiers de définitions d'objets (Section 2.5.5) - Ctrl+Alt+S

Ouvrir un fichier de projet - Ctrl+O

Commande Imprimer (Section 2.5.6) - Ctrl+P

Sauver-sous un autre fichier - Ctrl+A

Sauver le projet en cours - Ctrl+S
